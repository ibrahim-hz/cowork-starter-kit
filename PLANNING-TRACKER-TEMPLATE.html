Connect to JSONbin cloud storage

Save and load your tracker data through JSONbin.io so it's accessible from any browser and from Claude Cowork without using the Anthropic API.

  1. Sign up at jsonbin.io (free).
  2. Click your name → API Keys → copy the X-Master-Key.
  3. Paste it below. Leave Bin ID empty on first setup — we'll create one for you.
